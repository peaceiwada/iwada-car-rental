# Iwada Car Rental - Car Rental Web Application

A modern, professional car rental web application built with Next.js 14, Tailwind CSS, and React. This application allows users to browse cars, filter by various criteria, book vehicles, and manage their bookings.

## 🚀 Live Demo

[Deployed on Vercel](https://iwada-car-rental.vercel.app)

## 📋 Features

### Core Features
- ✅ **Homepage** with featured cars and hero section
- ✅ **Car Listing Page** with advanced filters (price, location, type, seats, transmission)
- ✅ **Car Details Page** with comprehensive vehicle information
- ✅ **Booking System** with date selection and price calculation
- ✅ **User Authentication** (Login/Register)
- ✅ **User Dashboard** to view and manage bookings
- ✅ **Contact Page** with form and map integration

### Problem-Solving Implementations
- ✅ **Optimized Page Loading Speed** - Image optimization, code splitting, and lazy loading
- ✅ **Empty States** - Graceful handling when no cars or bookings are available
- ✅ **Double Booking Prevention** - Server-side validation to prevent overlapping bookings
- ✅ **Clean & User-Friendly UI** - Modern design with smooth animations

### Bonus Features
- ✅ **Mock API** - Complete API endpoints for cars, auth, and bookings
- ✅ **Loading Skeletons** - Smooth loading experience
- ✅ **Animations** - Fade-in, slide-up, hover effects
- ✅ **Mobile-First Design** - Fully responsive across all devices

## 🛠️ Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS
- **Icons**: React Icons
- **Notifications**: React Hot Toast
- **Date Handling**: date-fns
- **HTTP Client**: Native Fetch API
- **Deployment**: Vercel
